# macwlt

Self-custodial cold wallet on MacOS Secure Enclave.

Why buying a $400 [Ledger Wallet™](https://www.ledger.com/) to store your keys when you already have a $2000+ [Apple](https://www.apple.com/it/macbook-pro/) one right in front of you?
